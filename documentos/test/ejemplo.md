# Ejemplo de Documento Markdown

Este es un documento de ejemplo para demostrar las capacidades del editor.

## Características Principales

- **Negrita** para énfasis
- *Cursiva* para destacar
- `Código inline` para fragmentos de código

## Edición Colaborativa

Este editor ahora soporta edición en tiempo real con WebSocket. Cuando múltiples usuarios editan el mismo archivo, los cambios se sincronizan automáticamente.

## Fórmulas Matemáticas

Puedes incluir LaTeX inline como $E = mc^2$ o en bloque:

$$
\int_{a}^{b} f(x) dx = F(b) - F(a)
$$
